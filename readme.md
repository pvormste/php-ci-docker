# php-ci

A php docker image which can be used for CI.

Docker Image: [pvormste/php-ci](https://hub.docker.com/repository/docker/pvormste/php-ci)

## Features

 - composer
 - ext-zip
 - ext-xml
 - ext-gd
 - ext-iconv
 - ext-pdo
 - ext-pdo-mysql
 - ext-psql
 - ext-pdo-psql
